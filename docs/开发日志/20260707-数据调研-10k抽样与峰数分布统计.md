## 4. 阶段 B：晶系分布（`atom_num<25` 子集，n=195,682）

| 晶系 | 20 万池内计数 | 占比 |
|---|---:|---:|
| tetragonal | 54,658 | 27.9% |
| cubic | 46,624 | 23.8% |
| monoclinic | 31,702 | 16.2% |
| orthorhombic | 30,534 | 15.6% |
| trigonal | 21,991 | 11.2% |
| hexagonal | 6,583 | 3.4% |
| triclinic | 3,590 | 1.8% |

**观察**：
- 池内晶系分布**高度不均衡**（tetragonal/cubic 占半数以上；三斜仅 1.8%）
- 对称性分析 **0 失败**（symprec=0.01）；个别 spglib 警告不影响 `get_crystal_system()` 返回
- D10 要求的「晶系尽量均匀」**不能依赖自然分布**，必须分层抽样——本轮已落实

## 6. `max_peaks` 候选建议（§9.3，待 PM 拍板）

基于 20 万池 `peak_num_filtered` 分布与 attention O(N²) 成本权衡：

| 候选 | 覆盖（20 万池） | 说明 |
|---|---|---|
| **51** | ~p95 | 偏保守，训练更稳；10k 样本约 **5%** 会被截断（p95=64） |
| **75** | ~p99 | **均衡默认**；10k 样本约 **1%** 会被截断（p99=91） |
| **543** | 100%（池 max） | 不截断，但长尾样本 attention 成本极高，不推荐作初版 |

**Agent 倾向**：10k smoke 先用 **75**；若显存/速度仍紧再降到 **51**。最终由 PM 决定。

## 8. 已知局限与风险

1. **20 万池采样误差**：全量 608 万中稀有晶系比例可能有偏差；但 10k 分层已从池内强制均衡，对 smoke 实验影响有限。
2. **symprec=0.01**：本轮默认提议，非历史项目约束；若与 MP100 评测口径不一致，后续可对比 0.1/0.01 敏感性。
3. **primitive 六参数角度**：部分立方/三方样本在 primitive 表示下出现 α=β=γ=60° 等（pymatgen 约化胞惯例），训练时应用 D9 晶系 mask 回归，不直接监督这些确定性角度。
4. **PXRD 与标签口径**：输入谱仍来自 conventional reduced 模拟，标签为 primitive（D1 已接受此风险）。
