## 结论

1. **仅把物理量塞进 32d/2L peak-token Bert 不够**——角色修正 / Q token 全部失败。  
2. **inverse_d2 直方图 bag 特征 + MLP 可过 G0**（复现并正式化 D-A3）。  
3. max-pool 略优于 sum-pool（minCS 96% vs 91%）。  
4. R1 主线改为 **histogram encoder**；peak-token Transformer 需更大容量/不同聚合，暂不作为 G1 主候选。
