## 裁决

1. **通过**：FOM Top-1 elementwise 从 **0% → 14.79%**，接近 pool 召回 16.5%。半胞偏好的根因确认是排序键里的「更小体积优先」。
2. **推荐默认推理**：`ref_volume = NN 预测体积` + `max_log_volume_ratio ≈ log(2)`；尺度变体默认关闭可再抬 pool +1.2pp（17.71%），但 FOM Top-1 略降（14.07%）。
