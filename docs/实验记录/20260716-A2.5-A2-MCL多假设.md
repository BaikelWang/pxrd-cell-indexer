## 结论

1. **本批真正的赢家是 A2-ctrl，不是 MCL。** `cs_conditional`（7 独立头，L=6）相对 A1b(FiLM) **+6.5pp**，过硬 Gate；低对称（tet/hex/trig/mono）全面抬升。
2. **MCL 正式指标失败**：A2a/A2b 均低于 A1b 与 ctrl；angMAE 崩到 ~10°，loose 掉到 ~60%。
3. **oracle_topk 高（48–53%）但 hyp-0 低（31/27%）** → 池内有更好候选，但当前「固定输出 hyp-0」无法兑现；MCL 制造了分散解，却伤害了主解精度。在未加可靠假设选择器前，**不采纳 MCL 作生产默认**。
4. **新 baseline 候选**：**T48-geom + cs_conditional + L=6**（A2-ctrl），取代 A1b(FiLM)。

## 对后续启示

- **A3 部分已由 ctrl 回答**：在 Peak Transformer + L=6 上，`cs_conditional` ≫ FiLM；`shared` 仍可补测，但优先级下降。
- **MCL 线关闭或降级**：除非先做 hyp 选择（置信度 / FOM 选 K），否则不要继续扫 K。
- **下一批优先**：B2 峰数、B1 A-adv、或在 **新 baseline=A2-ctrl** 上做容量/损失类杠杆；不要再挂在 FiLM 上比。
