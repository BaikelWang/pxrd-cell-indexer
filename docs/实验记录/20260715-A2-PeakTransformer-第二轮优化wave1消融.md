# A2 Peak Transformer 第二轮优化 · Wave-1 消融（3500 benchmark）

日期：2026-07-15
基座：T1（T20-pos，linear-g Fourier×16，cls_mean，FiLM head，matrix6，baseline SmoothL1）
对照：T1=10.14% strict elem（3500 胜出者）、E1c=7.14%（线性 baseline）
协议：3500 上单变量消融，best_metric=`strict_raw_top1_elementwise_rate`，100k 仅用于胜出者 lock-in。

## 动机（理论/数学分析）

对 encoder 与回归头做理论审视，提出并验证以下低风险、单变量假设：

| 臂 | 改动 | 针对的理论弱点 |
|----|------|----------------|
| P1 | Fourier on **log g**（替换线性 g） | 线性 g 给绝对分辨率；strict 要长度相对 5%（=log g 绝对分辨率），且晶胞缩放在 log g 上是平移不变 |
| P2 | **log+linear** 混合 Fourier | P1 的兜底，同时保留绝对/相对两种分辨率 |
| P3 | **attention pooling**（可学 query 掩码 softmax） | cls_mean 的均值项稀释「首峰主导」 |
| H1 | +joint 物理损失（相对长度+绝对角度 Huber，λ=0.1） | matrix6 分量空间 SmoothL1 与 north-star 度量不一致 |
| H2 | strict_hinge（对超过 ltol=0.05/atol=3° 的部分加铰链，λ=0.3） | strict 是合取指标，SmoothL1 优化均值而非最差维 |

## 结论

1. **没有一个 wave-1 臂在 strict elem 上超过 T1。** 除 H2 外，各臂落在 8.57–10.14% 区间，跨度 ~1.5pp，落在 T1 自身的 epoch 间噪声带内（T1 best 10.14 vs last 9.57，±0.57pp）。3500 规模对这些「表征/损失微调」是噪声受限的，不足以区分。
2. **H2（strict_hinge）明显有害**（strict 4.14%、clfCS 39.6%）：铰链项在通过样本上梯度为 0、只推违规样本，破坏了早期优化稳定性。**淘汰。**
3. **P1/P2 log-Fourier 未兑现理论收益**：3500 规模下 16 个线性频率已够用，相对分辨率的好处只在大晶胞长尾体现，样本太少无法体现。**暂不采纳**（可在 100k 复核）。
4. **P3 attention pooling 在 loose(50.0%)/clfCS 略优、strict 略差**：边际，非明确胜出。**暂不采纳。**
5. **最重要的诊断：精度墙在回归本身，不在 CS 路由。** oracleSE(10.57%) 仅比 predSE(10.14%) 高 0.43pp —— 即便给完美 CS，strict elem 也只有 ~10.6%。CS 分类 47% 虽低，但对最终 strict 的边际影响很小。`cs_correct 子集 21.58%` 是「可分类样本本身更易回归」的选择效应，不代表路由收益。

**保持 T1 架构不变。** 第一波低风险表征/损失微调是一个诚实的负结果：瓶颈不在 Fourier/pooling/损失度量，而在**回归精度上限**。

## 下一步方向（更高价值的结构性杠杆）

针对「回归精度墙」，deferred 的两条高上限杠杆最值得做：

- **H-sym**：按晶系自由度参数化输出（立方 1、四方/六方 2、正交 3、单斜 4、三斜 6），对称性由构造精确满足，减少高对称系统的无谓回归方差。需处理 Niggli 约化基下立方 F/I 的 60°/109.47° setting。
- **A-adv**：物理相对注意力偏置 b_ij=φ(g_i−g_j)，让 attention 直接以峰对间距为条件，显式利用倒易度规的差分结构。

（也可考虑在 3500 上对候选臂跑多 seed 以压噪声，但更建议直接投入结构性改动。）
