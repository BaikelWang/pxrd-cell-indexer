### 解读

- **数据红利明确**：真解 Top-1 +6.5pp（> 决策门 +5pp 阈值），晶系 acc +7.6pp，真解入池 +5.8pp。
- **分类改善显著**：「晶系错、晶格对」和「两者都错」均下降。
- **回归瓶颈仍在**：「晶系对、晶格错」仍占 **37.5%**（甚至略高于 10k 的 36.4%），hex/trig proxy match 仍接近 0。
- **排名问题未解**：真解在池但非 rank-1 仍 ~50%，Top-K 重排不是本阶段重点。

## 5. D29 决策

| 判定 | 结论 |
|---|---|
| joint +5pp 以上？ | **是（+6.5pp）** |
| **决策** | **数据 scaling 有效 → M2 主线改为冲更大规模（500k / 全量）** |
| 并行项 | hex/trig 回归架构改进仍值得做，但不阻塞 scaling |
