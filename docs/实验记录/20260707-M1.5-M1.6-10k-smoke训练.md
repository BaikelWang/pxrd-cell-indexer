## 4. 误差分析（原则 §11）

1. **晶系**：cubic 回归/匹配最好（Top-1 52–63%）；hexagonal/trigonal 几乎无 match，是主要短板。
2. **峰数/晶胞**：hexagonal primitive 常 `a≠b`（M1.2 已观测），全 6 参数回归未引入额外归纳偏置，可能需更多 epoch 或后续 primitive 约束消融。
3. **冻结 vs 解冻**：unfrozen 在 lattice MAE 与 Top-1 match 上略优于 frozen，但差距不大；10k 小样本下解冻未明显伤害晶系 acc。

## 5. 结论与下一步

- **Pipeline 端到端跑通**：encoder → heads → loss → train/valid → metrics → checkpoint。
- **打败信号地板**：三类 smoke 均在晶系 acc 与 lattice 指标上显著优于地板。
- **下一步（M1.7）**：Top-20 候选生成、完整 pymatgen `StructureMatcher` 评测、MP100 对照；可选 primitive 条件化 mask 消融（D25 后续）。
