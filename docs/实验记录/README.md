# 实验记录

> **用途**：存放**各次实验**的设置、过程、结果与分析，供日后回顾与复现。  
> **习惯**：每次跑实验后在此留档；原始大文件（checkpoint、log）放 `results/`，此处写摘要与链接。

## 写什么

每次实验建议包含：

1. **目的** — 验证什么假设
2. **设置** — 数据、模型、超参、命令、环境
3. **过程** — 训练/评测步骤、异常与调整
4. **结果** — 指标表、关键数字、与基线对比
5. **分析** — 结论、失败原因、下一步

## 命名建议

```
实验记录/
├── README.md
├── 20260706-exp001-baseline-smoke/
│   ├── config.yaml          ← 或 settings.md
│   ├── run.md               ← 命令与过程
│   └── analysis.md          ← 结果与分析
└── ...
```

- 目录名：`YYYYMMDD-expNNN-简短描述/`
- 或用单文件：`20260706-exp001-baseline-smoke.md`（小实验）

## 与开发日志的分工

| | 开发日志 | 实验记录 |
|---|---|---|
| **粒度** | 工作进展、决策、方案 | 单次实验全流程 |
| **读者** | 项目成员、周报 | 未来复现与对比 |
| **示例** | 「决定用 241113 主集训练」 | 「exp003：241113 train 10 epoch，lattice match 42%」 |
