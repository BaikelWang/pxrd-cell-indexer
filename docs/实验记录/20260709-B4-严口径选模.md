## 1. 假设

训练 best ckpt 盯宽松 `0.3/10°`，与北极星 `0.05/3°` 不对齐；改为严口径选模可选出更接近硬指标的权重。

---

## 2. 实现

| 改动 | 说明 |
|---|---|
| `TrainConfig` | `eval_ltol/atol` + `strict_ltol/atol`；`best_metric` 支持 `strict_raw_*` / `strict_composite` |
| `trainer.py` valid | 每 epoch 双报 loose + `strict_raw_top1_{lattice_match,elementwise}_rate` |
| warm_start | **仅 continuous** 跳过 `embed_positions`（修 A3/B4 误跳 discrete pos） |
| config | `configs/scale_100k_b4_strict_select.yaml` |

训练：100k，warm_start baseline，8 epoch，`best_metric=strict_raw_top1_elementwise_rate`，best@**ep7**（5.64%）。

## 5. 结论

1. **双轨监控可用**：后续所有训练默认应写 `strict_*` 字段。  
2. **选模对齐有小收益**：valid 严口径 elementwise **+2pp**（3.6%→5.6%），方向正确。  
3. **远未达 Gate-2**（fom≥30% / MP100≥35%）；MP100 fom 甚至略降。  
4. **仅改选模不够**：必须叠加 **B1 严口径 loss**（角度 hinge / 加权）才能压 `angle_mae`（仍 ~9.8°）。  
5. warm_start 修复：discrete 路径现可完整加载 42 keys。
